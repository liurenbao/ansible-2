0.k8s 创建rtd namespace
kubectl create namespace rtd

1.部署zk
！！！先修改values.yaml中的zookeeper镜像地址，用户harbor中有zookeeper的地址！！！
cd zookeeper
helm upgrade --install rtd-zookeeper -n rtd
2.部署rtd-web
cd rtd-web
helm upgrade --install rtd-web -n rtd
3.部署rtd-ruleengine
cd ruleengine
helm upgrade --install rtd-ruleengine -n rtd
4.部署rtd-workflow
cd workflow
helm upgrade --install rtd-workflow -n rtd

注意：
1.如果遇到部署错误，需要检查部署的各种yaml文件，调试命令:
cd 对应helm文件夹
helm install --dry-run --debug rtd-XXX  .
该命令会将所有部署yaml进行渲染，并打印输出，且不会部署

2.如果在非集群服务器上部署，需要将集群kubeconfig文件下载并替换helm文件夹中的kubeconfig文件。
部署命令更改为：helm upgrade --install rtd-XXX -n rtd --kubeconfig=kubeconfig
